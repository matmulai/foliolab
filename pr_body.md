💡 **What:** Added descriptive `aria-label` attributes to the icon-only "✕" (Close) buttons and "←" (Back) buttons within the Data Sources page.
🎯 **Why:** Previously, these buttons contained only text symbols or unicode icons without labels. Screen readers would either announce confusing unicode character names or remain silent, preventing visually impaired users from understanding the function of these actions. This enhancement ensures the button's intent is communicated properly.
📸 **Before/After:** Visually identical, but semantically accessible.
♿ **Accessibility:** Added contextual labels like "Close form", "Back to Home", and "Back to GitHub Selection" to make the interface fully navigable and understandable for screen reader users.
